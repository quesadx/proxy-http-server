# Plan 02-02: HTTP Domain and Keyword Filtering

## Objective

Wire the blocklist check into the HTTP request path. Modify `handler.py` so that `handle_http()` checks the `Host` header domain against the blocklist **before** forwarding. If blocked, return the 403 block page. If allowed, forward normally.

**Purpose:** HTTP domain filtering is the first and most common filtering path — every HTTP request through the proxy goes through `handle_http()`.

**Output:** Modified `src/handler.py` with blocklist-aware `handle_http()`.

    Modify `src/handler.py`:

    1. **Add import** at the top of `handle_http()` method (alongside the existing lazy import of `forward_request`):

         ```python
         from src.filter import blocklist, BLOCK_PAGE
         ```

    2. **Add Host header extraction and blocklist check** before the `forward_request()` call. The check must:
       - Get the `host` value from `headers.get("host", "")`
       - If host is present, strip the port: `host.split(":")[0]`
       - If `host` is non-empty AND `blocklist.is_blocked(host)` returns True:
         - Format the block page: `response = BLOCK_PAGE.format(domain=host)`
         - Write to wfile: `self.wfile.write(response.encode("utf-8"))`
         - Flush: `self.wfile.flush()`
         - Return early (do NOT call forward_request)
       - If host is empty OR not blocked, fall through to the existing `forward_request()` call unchanged

    3. **Keep the existing lazy import** for `forward_request` — it remains `from http_relay import forward_request` (not `from src.http_relay`). The original code uses a bare import because `handler.py` is run from the project root. Preserve this pattern.

    4. **Preserve all existing code** for non-blocked requests: the method must behave identically to the original for allowed domains.

    The modified `handle_http()` flow becomes:
    ```
    handle_http(request_line, headers, body):
      from src.filter import blocklist, BLOCK_PAGE  ← NEW
      host = headers.get("host", "").split(":")[0]  ← NEW
      if host and blocklist.is_blocked(host):         ← NEW
        self.wfile.write(BLOCK_PAGE.format(...))     ← NEW
        self.wfile.flush()                           ← NEW
        return                                       ← NEW
      from http_relay import forward_request         ← EXISTING
      forward_request(...)                           ← EXISTING
    ```
