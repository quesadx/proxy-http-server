    3. **`Blocklist` class** implementing:
       - **Class constant** `POLL_INTERVAL = 2`
       - **`__init__(self, filepath: str = "config/blocked_domains.txt")`**:
         - Store `self._filepath = Path(filepath)`
         - Create `self._lock = threading.Lock()`
         - Initialize `self._domains: set[str] = set()`
         - Initialize `self._last_mtime: float = 0.0`
         - Call `self._load()` then `self._start_watcher()`
       - **`_load(self) -> None`**:
         - Read file with `self._filepath.read_text(encoding="utf-8")`
         - Split into lines, strip each line, skip empty lines and lines starting with `#`
         - Add remaining lines (lowercased) to a local `set`
         - If `FileNotFoundError`, silently use empty set (no blocklist file is not a crash)
         - Replace `self._domains` atomically under `self._lock`
       - **`_start_watcher(self) -> None`**:
         - Define inner `_watcher()` function with `while True: time.sleep(POLL_INTERVAL)`
         - Inside loop: try `os.path.getmtime(self._filepath)`, skip on OSError
         - If mtime differs from `self._last_mtime`, update `self._last_mtime = mtime`, call `self._load()`
         - Create `threading.Thread(target=_watcher, daemon=True)` and start it
       - **`is_blocked(self, domain: str) -> bool`**:
         - Lowercase the input domain
         - Under `self._lock`:
           - Check `domain in self._domains` (exact match) → return True
           - Iterate patterns: if `fnmatch.fnmatch(domain, pattern)` → return True
         - Return False if no matches
       - Source: RESEARCH.md Pattern 1 lines 164-218 for the complete implementation.

    Python stdlib only. No external packages. Follow the pattern and implementation from RESEARCH.md exactly — it has been verified against Python 3.13.13.

    1. **`extract_sni(data: bytes) -> str | None`** function:
       - Signature: `def extract_sni(data: bytes) -> str | None:`
       - Docstring: `"""Extract SNI hostname from raw TLS ClientHello bytes. Returns hostname (lowercased) or None if not found/unparseable."""`
       - Implementation (fleshed out from RESEARCH.md Pattern 2 lines 261-333):
         - Guard: if `len(data) < 5 or data[0] != 0x16` → return None (not a TLS Handshake record)
         - Guard: if `len(data) < 43` → return None (minimum ClientHello size)
         - Position cursor `pos = 43` (skip: 5-byte record header + 4-byte handshake header + 1-byte version + 32-byte random)
         - Read session_id_len = data[pos], advance pos by `1 + session_id_len`
         - Guard: if `pos + 2 > len(data)` → return None
         - Read cipher_len as big-endian uint16 via `struct.unpack("!H", data[pos:pos+2])[0]`, advance pos by `2 + cipher_len`
         - Guard: if `pos + 1 > len(data)` → return None
         - Read comp_len = data[pos], advance pos by `1 + comp_len`
         - Guard: if `pos + 2 > len(data)` → return None
         - Read ext_total_len as big-endian uint16, advance pos by 2, set `ext_end = min(pos + ext_total_len, len(data))`
         - Walk extensions in a while loop (`pos + 4 <= ext_end`):
           - Read ext_type (uint16 BE), ext_len (uint16 BE)
           - Advance pos by 4, guard `pos + ext_len > ext_end` → break
           - If ext_type == 0x0000 (server_name):
             - sni_start = pos + 3 (skip 2-byte SNI list length + 1-byte name_type)
             - Guard: if sni_start + 2 > pos + ext_len → break
             - Read name_len (uint16 BE)
             - sni_start += 2
             - If sni_start + name_len <= pos + ext_len:
               - Return `data[sni_start:sni_start + name_len].decode("utf-8", errors="replace").lower()`
           - Advance pos by ext_len
         - Return None if no SNI found
       - This function accepts `bytes`, NOT a socket — separation of I/O from parsing makes it testable.
       - Source: RESEARCH.md Pattern 2 lines 261-333 and the function at lines 612-657.
